# 网页（HTML）中的特殊字符
https://www.cnblogs.com/aspirant/p/3571374.html

```
&#D; 其中D是一个十进制数值

下面是一些特殊字符的数字参考示例：

&#169; 对应于特殊字符“©”

&#174; 对应于特殊字符“®”

所谓实体参考，实际上就是用有意义的名称来表示特殊字符，通常由前缀“&”，加上字符对应的名称，再加上后缀“;”而组成，其表达方式如下：

&name; 其中name是一个用于表示字符的名称，它是区分大小写的

例如，可以使用“&copy;”来表示版权符号“©”，用“&reg;”来表示注册商标符号“®”，很显然，这比数字要容易记忆得多。

```
