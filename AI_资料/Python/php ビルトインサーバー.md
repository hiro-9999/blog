```
 ビルトインサーバーを立ち上げるコマンドを入力します。

$ php -S localhost:8080 -t ~/Desktop/workspace

http://localhost:8080
 

各コマンドの意味はこちら。

$ php -S localhost:8080 -t ~/Desktop/workspace

-S
ビルトインサーバーを立ち上げるオプション。
localhost:8080
自分のローカル環境の8080ポートを使用する指定。
-t
ドキュメントルートの指定。
-tの後にドキュメントルートのパスを指定できます。
```
