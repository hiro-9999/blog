 3 Firebase Cloud Messaging（FCM）
Firebase Cloud Messaging はアプリケーションに対してメッセージ（プッシュ通知）を無料で送受信できる機能です。
iOS や Android など異なるデバイス間でも送受信ができ、セグメントごとにメッセージを送信することも可能です。
送信したメッセージはダッシュボード上でモニタリングもできます。
