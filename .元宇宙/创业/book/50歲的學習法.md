# “學習的主要目的，不是要得到答案，而是要知道有各種各樣的答案。”


```
“重要的不是知識本身，而是如何以自己的方式掌握知識。
現在的答案，未必是十年後的答案。
要深入且廣泛地探索事物，養成尋找其他解答的習慣。
學習不是為了知道問題唯一的解答，而是為了知道有各種答案。
釐清自己學習的目的—
是為了得到報酬？還是要鑽研嗜好？
是為了在目前的工作更上層樓？或者是想離開組織獨當一面？
目的是貢獻社會嗎？或是希望藉由學習與異性等其他人交流？
將英語視為與外國人交流（說話），或收集資訊（閱讀）的工具來學習。
重新學習正確的母語。
廣泛閱讀，並使用部分熟讀法。”

“一開始，就先將輸出所學當作目的，在這個前提下吸收、學習。
如此一來，就能根據之前的統整，有效率地收集資訊。”

“不是要藉由吸收知識的過程來學習，而是要靠著輸出的過程來學習。
使用知識，能加深記憶。
積極尋找機會發表所知，例如在網路上發文，或是在讀書會上發言。
刻意發表會招致反對意見的內容。
美國人擅長演說的祕訣是，事先擬定講稿並認真演練。
在日常對話中，盡力讓對方樂在其中。
與新朋友結緣，是發表所知的最大回報。”

“養成習慣，思考其他人不會思考的事。
結果比過程重要。要著眼於長期結果，而非短期結果。
我們應該維持的，不是身體的運動機能，而是活力。想活化大腦，學習比運動更有效。
不以追求更高的工作職位為目標，不要選錯人生的終點
退休後還有人願意與自己往來，人生才算豐富。用來讓別人開心的錢不可小氣。”
```
