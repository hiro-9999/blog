因此，数组中的全部元素的异或运算结果即为数组中只出现一次的数字。

JavaC++Python3C#Golang

class Solution {
    public int singleNumber(int[] nums) {
        int single = 0;
        for (int num : nums) {
            single ^= num;
        }
        return single;
    }
}
