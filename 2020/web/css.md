--styleオプションで変換方法を変える

変換方法は主に4つ。
nested, expanded, compact, compressed

sass style.scss:style.css --style nested

## --watchオプションで自動コンパイルする

毎回ファイルを更新する度にコンパイルするのも面倒なので自動化しましょう。
sass --watch style:scss:style:css
